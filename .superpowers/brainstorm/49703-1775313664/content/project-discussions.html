How It Works Together

Task Comments

For conversation about that specific task. "Do we have a 5th monitor?" lives on the monitor task.

Project Discussions

For conversations that span multiple tasks or are about the project broadly. Link related tasks inline.

Decisions

Any comment (on a task or in a discussion) can be marked as a decision. All decisions surface in one place.

Nothing Gets Lost

Unread badges on tasks + discussions. @mentions push notify. Decisions tab is the permanent record.
